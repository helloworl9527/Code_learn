# 题目
## 解法1 
```
number = 10020  
w = 10000  
count = 0  
for i in range(5):  
    x = number // w  
    if x:  
        count += 1  
        number = number % w  
    elif count > 0 and x == 0:  
        count += 1  
        number = number % w  
    elif x == 0:  
        w = w // 10  
        continue  
    w = w // 10  
    print("number：{},几位数:{}".format(x, count))
```
